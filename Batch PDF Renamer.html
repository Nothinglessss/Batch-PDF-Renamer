Batch PDF Renamer

Automatically renames text-based PDF files based on its title! New file will be generated under the subfolder "new" of the current directory.

NOTE:You will need python and the repos "pdfminer.six","mojimoji","pybase64" and "regex" to use this application. If you do not have the repos intalled, type "pip install (repo name as shown above)" into your command prompt window to install.
